## Description

API Rest made with [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

###### Create a .env file
```
DB_USER='your mongodb atlas user'
DB_PASSWORD='your mongodb atlas user password'
DB_HOST='your mongodb cluster'
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
